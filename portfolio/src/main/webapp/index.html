My Projects

6502 CPU Emulator

An emulator of the 6502 processor written in C++, designed with a modular API for integration with 8-bit game console emulation. The code uses a functional approach to instructions, allowing dozens of instructions to be implemented with only a few functions. The project is structured to be independently useful for educational purposes.

Bing Schedules

A website allowing Binghamton University to input their class schedule and share them with custom groups. Course input uses a custom formatted database of classes, allowing for quick lookups, keyboard-only or mouse-only input, and automatic suggestions for labs and discussions.

Github

CHIP-8 Emulator

An emulator of the CHIP-8 educational processor written in C++. It features a debug panel allowing for both recreational and educational applications. Also contains a flexible, standalone disassembler to allow for debugging.
